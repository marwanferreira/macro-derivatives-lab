# Macro Derivatives Lab

Pre-trade research, pricing tools, and market analysis for the FINC3012 CME Trading Simulator assignment.

## Objective
This repository is used to research, evaluate, and compare several derivatives trading strategies linked to the current macro environment before execution in the CME Trading Simulator.

## Candidate strategies
1. Treasury straddle
2. Treasury butterfly
3. Gold / silver ratio trade
4. WTI crude oil call spread
5. Silver call spread

## Repo structure
- `data/` raw and processed market data
- `notebooks/` research notebooks by strategy
- `pricers/` pricing and payoff tools
- `research/` market view and trade memos
- `utils/` helper functions
- `outputs/` charts, tables, and trade sheets

## Workflow
1. Build market view
2. Run pre-trade checks
3. Compare strategies
4. Execute the best setup(s)
5. Document outcomes for the final report
